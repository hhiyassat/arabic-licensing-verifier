# الخلاصة التمهيديّة — الصندوق التحقيقيّ

> **وظيفة هذا الملفّ:** مدخلُ قارئٍ للمشروع، لا مادّةٌ دستوريّةٌ مُلزِمة. يُلخِّص روح الصندوق
> ومساره في نثرٍ متّصل. **لا يُعدّل الدستور ولا بصماته**؛ المرجع المُلزِم يبقى
> `constitution.md` المُجمَّد. عند أيّ تعارضٍ ظاهر، الدستور هو الحاكم.

---

## ما هو الصندوق التحقيقيّ

الصندوق التحقيقيّ ليس مولّدًا للمعنى، بل **محكمةُ ترخيصٍ للحكم المُدَّعى**. لا يبدأ بإنتاج حكمٍ
جديد، ولا يدّعي اكتشاف المعنى من تلقاء نفسه، بل يستقبل دعوى محدّدةً ذات نوعٍ وحمولة، ثم يحاكمها
عبر بواباتٍ مرخِّصة.

يسأل الصندوق عن الحكم المُدَّعى:

هل بنيته مقروءة؟ هل نوعه وحمولته متطابقان؟ هل دليله كافٍ؟ هل رتبته لا تتجاوز دليله؟ هل أثره
محفوظ؟ هل نسخته معلومة؟ هل بقاياه ظاهرةٌ ومصنَّفة؟ هل مساره خالٍ من القفز؟

---

## المصبّات الأربعة

فإن اكتملت البوابات، ولم يظهر مانع، ولم تبقَ بقايا مفتوحة، انتهى الحكم إلى:
```
Licensed
```
وإن ظهر مانعٌ قادح، انتهى إلى:
```
Blocked
```
وإن نقصت بوابةٌ أو غاب دليلٌ لازم، انتهى إلى:
```
Deferred
```
وإن بقي أثرٌ غير حاسمٍ لكنه محفوظٌ ومصنَّف، انتهى إلى:
```
Residual
```

---

## القواعد الحاكمة (منقولة عن الدستور)

ولا يبلغ `GenericPayload` رتبة `Licensed`. ولا رتبةَ بلا دليل. ولا نتيجةَ بلا أثر. ولا مخرجَ بلا
نسخة. ولا حكمَ نهائيٍّ مع قفزةٍ غير مرخّصة.

---

## مسار الصندوق

وبذلك تكون وظيفة الصندوق:
```
JudgmentClaim → Payload → Evidence → Trace → VersionStamp → Sink
```
لا بوصفها سلسلة توليد معنى، بل بوصفها **سلسلة تحقيقٍ وترخيص**.

---

## ما أُنجز من الحراسة

وقد أُنجزت حراسةُ الانتقال وحراسةُ التقسيم: فُصل الدستور عن الخطة، وفُصلت الخطة عن تقرير التنفيذ،
وفُصل مستوى الحكم على العربية عن مستوى الحكم على سلامة الآلة.

---

## الخطوة التالية عند فتح التنفيذ

إنشاء الصندوق التحقيقيّ نفسه على هذا المسار:
```
JudgmentClaim → Payload → Evidence → Trace → VersionStamp → Sink
```

أمّا حزم التحقيق اللاحقة (الباعث، الخانة، المؤشّر، الوزن، الجنس، النسبة، المنطوق، المفهوم،
الحكم) فأفقُ توسّعٍ مستقبليٌّ مُثبَّتٌ في `development_plan.md` — لا جزءٌ من الدستور المُجمَّد،
وتدخل بوصفها حزمَ تحقيقٍ **تُطبّق** الدستور لا تُعدّله.
